# Accounting Batch 4A

A Next.js web app for Accounting Batch 4A to upload and manage group presentations (PPT/PDF).

## Features
- Upload group presentations with topic and members  
- Search and view submitted groups  
- Download or delete submissions  
- Responsive UI: table on desktop, cards on mobile  
- Uses Cloudinary for file storage  

## Tech Stack
Next.js 15, React 19, TailwindCSS 4, MongoDB (Mongoose), Cloudinary, react-hook-form, yup, axios
